🛠 Arabic / Persian Text Shaper — Guide

What is this tool?

Arabic and Persian text doesn't work in Unity UABE by default — the letters appear disconnected and reversed. This tool automatically reshapes each letter into its correct contextual form and reverses the string so Unity's RTL engine displays it perfectly.

Boxes explained
Input raw arabic

Paste or type your Arabic / Persian text. Supports full harakat (فَتْحَة، شَدَّة، تَنْوِين).

Lines reshaped only

Each line after letter-shaping. Verify it looks correct. Not yet reversed.

Reshaped & Flipped per line

Each line shaped + fully reversed (keeping diacritic clusters intact). What Unity needs per line.

Combined paste into uabe

All lines joined by your separator. Copy this directly into UABE. Also supports reverse import — paste game text here and click ↑ Import to edit it.

Settings

Box Width (px) — Pixel width of the in-game text box. Lines wrap to match exactly what appears in-game.

Font Size (px) — Must match your in-game font size for accurate wrapping.

Separator — Character(s) between lines in combined output. Default \n. The ? button beside it opens this guide.

Font File — Upload your game font (.ttf/.otf) for pixel-perfect line-break calculations.

Letter keyboard

Click ⌨ to open the built-in keyboard. Orange buttons are Persian-only letters (پ چ ژ گ ک ی). The Harakat row adds diacritics to any letter.

Two-way editing

Paste a previously exported game string into the Combined box, then click ↑ Import. The tool reverses and un-shapes it back to readable Arabic in the Input box so you can edit and re-export.

⇅ Reverse line order

The ⇅ button beside the separator field reverses the order of all output lines. When green, lines run from last to first (Line 3 → Line 2 → Line 1) across the Lines, Reshaped & Flipped, and Combined boxes. The Input and Preview are unaffected. Useful when your game engine reads lines bottom-up, or when a specific dialogue system expects reversed line order.

ما هي هذه الأداة؟

النصوص العربية والفارسية لا تعمل في Unity UABE بشكل افتراضي — الحروف تظهر منفصلة ومعكوسة. هذه الأداة تُعيد تشكيل كل حرف إلى شكله الصحيح حسب موقعه، وتعكس النص ليظهر بشكل مثالي في محرك Unity.

شرح كل صندوق
الإدخال نص خام

الصق أو اكتب نصك العربي أو الفارسي. يدعم الحركات الكاملة (فَتْحَة، شَدَّة، تَنْوِين).

الأسطر مُشكَّل فقط

كل سطر بعد التشكيل. تحقق من صحة النص بصرياً قبل النسخ. لم يُعكس بعد.

مُشكَّل ومعكوس لكل سطر

كل سطر مُشكَّل ومعكوس بالكامل مع الحفاظ على الحركات. هذا ما يحتاجه Unity لكل سطر.

مُدمج الصق في UABE

كل الأسطر مدموجة بالفاصل. انسخ هذا مباشرةً في UABE. يدعم الاستيراد العكسي — الصق نص اللعبة هنا واضغط ↑ استيراد لتعديله.

الإعدادات

عرض الصندوق (px) — عرض صندوق النص في اللعبة بالبكسل. تُقسَّم الأسطر تلقائياً لتطابق ما يظهر داخل اللعبة.

حجم الخط (px) — يجب أن يتطابق مع حجم الخط في اللعبة لضمان دقة كسر الأسطر.

الفاصل — الحرف بين الأسطر في المُدمج. الافتراضي \n. زر ? بجانبه يفتح هذا الشرح.

ملف الخط — حمّل خط اللعبة (.ttf/.otf) لحسابات دقيقة لكسر الأسطر.

لوحة الأحرف

اضغط ⌨ لفتح لوحة الأحرف المدمجة. أزرار برتقالية للحروف الفارسية فقط (پ چ ژ گ ک ی). صف الحركات يضيف الفتحة والضمة والكسرة وغيرها لأي حرف.

التعديل ثنائي الاتجاه

الصق نصاً مُصدَّراً مسبقاً في صندوق المُدمج، ثم اضغط ↑ استيراد. تقوم الأداة بعكس التشكيل وإعادة النص المقروء إلى صندوق الإدخال لتعديله وإعادة تصديره.

⇅ عكس ترتيب الأسطر

زر ⇅ بجانب حقل الفاصل يعكس ترتيب جميع أسطر الناتج. عند تفعيله يصبح أخضر، وتُعرض الأسطر من الأخير للأول (السطر 3 ← السطر 2 ← السطر 1) في صناديق الأسطر والمعكوس والمُدمج. صندوق الإدخال والمعاينة لا يتأثران. مفيد عندما يقرأ محرك اللعبة الأسطر من الأسفل للأعلى، أو عند نظام حوارات يتطلب ترتيباً معكوساً.
